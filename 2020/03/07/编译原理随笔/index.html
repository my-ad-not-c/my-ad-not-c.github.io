第三章 词法分析

1、正则表达式（RE）

  • 是一种用来描述正则语言的更紧凑的表达方式
  • 每个正则表达式r定义一个语言L（r）
  • 递归定义：
    • 空串ε是一个RE，L（ε）= { ε }；
    • 如果a属于∑，则a是一个RE，L（a）= { a }；
    • 假设r和s都是RE则
      • r | s是一个RE，L（r | s）=两语言并集
      • rs是一个RE，两语言乘积
      • r^*^是一个RE，语言的克林闭包
      • （r）是一个RE，原语言
  • 正则语言：可以用RE定义的语言，也叫正则集合
  • 正则文法与正则表达式的等价

2、正则定义

  • 如下形式
    • di 代表某个RE ri
    • 每个 di 都是一个新符号
    • 每个 ri 是 ∑ 并上 {di} 上的正则表达式

3、有穷自动机

  • 最长子串匹配原则
  • DFA
    • （S，∑，δ，s0，F）
    • δ：S×∑→S
  • NFA
    • （S，∑，δ，s0，F）
    • δ：S×∑→2^S^
  • 带有ε-边的NFA
    • （S，∑，δ，s0，F）
    • δ：S×（ ∑并{ε} ）→2^S^
  • 两种自动机的等价性
  • 带不带ε-边的等价性
