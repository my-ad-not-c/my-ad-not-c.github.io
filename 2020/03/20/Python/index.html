  5. 递归函数

    #内部调用自己就是递归函数
>>> def fact(n):
...     if n==1:
...             return 1
...     return n*fact(n-1)
...
>>> fact(2)
2
>>> fact(4)
24
>>> fact(100)
93326215443944152681699238856266700490715968264381621468592963895217599993229915608941463976156518286253697920827223758251185210916864000000000000000000000000
#递归函数要防止栈溢出
#函数调用通过栈实现，每进入一个函数调用，栈就会增加一层栈帧，每当函数返回，就会减一层栈帧，如果调用过多会溢出
#解决栈溢出的方法是通过尾递归优化，循环是一种特殊的尾递归
#尾递归指，函数返回时，调用自身本身，return语句不包含表达式，这样无论调用多少次，都只使用一个栈桢
>>> def fact(n):
...     return fact_iter(n,1)
...
>>> def fact_iter(num,product):
...     if num==1:
...             return product
...     return fact_iter(num-1,num*product)
...
>>> fact(3)
6
#遗憾的是，大多数语言并没有对尾指针做优化，包括Python，所以即使这样写，仍然会溢出

...             n=n+1
...     return 'done'
...
